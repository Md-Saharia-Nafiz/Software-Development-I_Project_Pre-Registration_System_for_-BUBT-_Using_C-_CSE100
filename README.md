
# Pre-Registration Course 2026 {Tri Semester} for (BUBT) 

This project is a Pre-Registration System for Bangladesh University of Business and Technology (BUBT). Using C++ and Object-Oriented Programming.


## üìÑDescription:

This project is a C++ OOP-based Pre-Registration System for BUBT students.
It simulates the pre-registration process for the 2026 Tri Semester.
The program displays a welcome message and course registration details step by step.
Students can input their personal and academic details through console prompts.
After completing the registration, it shows a confirmation message with instructions.
The program also includes Enter key navigation and Q/q to exit functionality.
## üí° Key Concepts Used:

(i) Object-Oriented Programming (OOP): Used Student class to structure the project.

(ii) Encapsulation: Protected student data with private variables and public methods.

(ii) Class & Object: Objects manage student registration and program flow.

(iv) Member Functions / Methods: inputDetails() and displayConfirmation() for modular code.

(v) Console Interaction: User-friendly prompts with Enter key navigation.

(vi) Exit Functionality: Q/q option to safely close the program.

(vii)Code Modularity: Organized functions for readability and future upgrades.

(viii) Step-by-Step Flow: Welcome ‚Üí Registration ‚Üí Confirmation ‚Üí Exit.
